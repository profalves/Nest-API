Nest API

API with Nest framework TypeScript.

Installation

$ npm install

Running the app

# demo
$ npm run start

# developing mode
$ npm run start:dev

# production mode
$ npm run start:prod

Test

# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

See API documentation

  • API routes documentation https://localhost:3000/api

  • General documentation
